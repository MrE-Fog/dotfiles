pbcopyfile(1) -- copy a file to the pasteboard
==============================================

## SYNOPSIS

`pbcopyfile` <var>file</var>

## DESCRIPTION

The pbcopyfile(1) command copies the specified file to the pasteboard (the Clipboard).
